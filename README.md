# LeetMetric

A modern, smooth LeetCode statistics viewer.

## Features
- **Visual Stats**: View Easy, Medium, and Hard problem progress with circular progress bars.
- **Submission Data**: Detailed breakdown of total and accepted submissions.
- **Dark Mode**: Sleek, modern dark interface.
- **Local Proxy**: Custom Node.js backend to bypass CORS restrictions when fetching LeetCode data.

## Prerequisites
- Node.js installed on your machine.

## Installation

1. Clone the repository or download the files.
2. Open a terminal in the project folder.
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the backend server:
   ```bash
   npm start
   ```
   You should see: `Server is running on http://localhost:4000`

2. Open `index.html` in your web browser.

3. Enter a LeetCode username and click **Search**.

## Technologies
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express
